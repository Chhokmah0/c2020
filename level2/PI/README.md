### 题目：PI（2014级陈飞宇大神挖坑）

### 要求：

1. 计算PI，精度>1万位（够黑啊！）
1. 打印总时间
1. 打印内存消耗的最大值

### 参考：

[2014级柱神作品](https://github.com/holdzhu/uestc2015ccpp/tree/master/Pi) （做完本题后再看啊！！！）

* 所用公式 pi/2 = 1 + 1/3 + 1/3 * 2/5 + 1/3 * 2/5 * 3/7 + ...  
* 优点：迭代可以使用之前计算过的数，代码好实现  
* 缺点：
    1. 收敛速度大概2~3次迭代精度才加1位；
    2. 因为要使用之前的数所以没法多线程
